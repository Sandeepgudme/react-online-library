# 📚 React Online Library

A React-based Online Library System that allows users to browse, search, filter, and manage books efficiently. Built using React, Redux, and React Router.
 ## 🌐 Live Demo
👉 https://sandeepgudme.github.io/react-online-library/

---


## 🚀 Features

- 📖 View list of books with images and details  
- 🔍 Search books by title  
- 📂 Filter books by category (Fiction, Non-Fiction, Sci-Fi)  
- 📄 View detailed information of each book  
- ➕ Add new books dynamically  
- 🔁 State management using Redux  
- ❌ 404 Page for invalid routes  

---

## 🛠️ Tech Stack

- React (Vite)
- Redux Toolkit
- React Router DOM
- CSS

## ⚙️ Installation & Setup

1. Clone the repository:

git clone https://github.com/sandeepgudme/react-online-library.git


2. Navigate to project folder:

cd react-online-library


3. Install dependencies:

npm install


4. Run the project:

npm run dev
